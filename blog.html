Question: Discuss the scope of var, let, and const.

Answer: Var is a global variable but nowadays we rarely use var. We are now using let and const variables. When we declare a variable outside the function it is called global scope. When we declare a variable inside a function it is called local scope or functional scope. Global variables we use anywhere but local or functional variables we cannot use anywhere.
